### 깃헙 컨벤션
패키지 및 파일명이 일관되도록 아래의 형식을 지켜주세요!😆  

</br>

**[computer-science]**
- 하위 패키지 명은 그 주 공부한 메인 주제로 정의합니다!  
ex) `운영체제`
- 개개인의 질문지는 상위 패키지 하위에 본인의 이름으로된 markdown 파일을 만들어서 올리도록 해요 :)  
ex) `운영체제/jiyoung.md`
- 커밋: [메인 주제]: [푸시하는 파일명.md]  
ex) `운영체제: jiyoung.md`

</br>

**그 외, 브랜치명과 풀리케 타이틀은 자유롭게 하도록 해요 :)**
